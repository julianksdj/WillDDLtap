# WillDDLcrossFB
 Multitap delay 
 Based on  Will Pirkle's Audio Processing book chapter 7.7 and 7.9.3
 Book "Will Prikle - Designing Audio Effect Plugins in C with DSP theory"
 Implemented using JUCE framework
 
 Features:
  - Delay time from 0 to 2000 ms
  - Feedback control
  - From 1 to 10 taps
  - Dry/wet control
